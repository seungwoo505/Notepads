# 개요

디지털 메모장은 여러 종류가 존재한다. 대표적으로 삼성과 애플에서 자체적으로 사용하는 삼성 노트와 메모장이 그 예시이다.
하지만 사용하는 제품이 다를 경우 공유하는 것이 제한적이게된다.
구글에서 개발한 keep이라는 메모장도 있긴하지만 구글 아이디가 요구가 된다는 단점이 있다.

그래서 고안한 것이 이 메모장이다.
메모장에 글, 그림 또는 사진을 넣고 변별할 수 있게 제목과 비밀번호를 입력하고 공유하고싶은 대상에게 제목과 비밀번호만 알려주면 상대방도 쉽게 접근할 수 있다.
그리고 실시간 통신을 통해 이미 작성된 메모장에 공유하고싶은 대상을 초대하고 같이 작성을 하여 멀리 있어도 의견 공유를 실시간으로 할 수 있다.
